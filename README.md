# Are you running?
This is our entry for [Mallorca Game Jam 2017](https://gamejam2017.gdgmallorca.com/) developed in 48h

Download binaries at [itch.io game page](https://syhid.itch.io/are-you-running)

Theme: Trumpocalypse

Made with LibGDX and written in Kotlin.
